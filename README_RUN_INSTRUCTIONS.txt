HOW TO RUN PENTA-PERSONA AI
===========================

Welcome! To run this project effortlessly, you have two options:

OPTION 1: Using Antigravity (Recommended)
-----------------------------------------
1. Open this folder in your code editor with Antigravity enabled.
2. In the AI chat, simply say:
   "Run project"
   
   The agent will automatically:
   - Check and install all dependencies.
   - Verify if Ollama is running.
   - Launch the application for you.

OPTION 2: Manual Run
--------------------
1. Open a terminal in this folder.
2. Run the calibration script:
   python calibrate_and_run.py

IMPORTANT NOTES
---------------
1. Ensure the `.env` file is present in this folder (it contains the API keys).
2. If you want to use the local LLM features, make sure Ollama is installed and running (`ollama serve`).

Enjoy!
